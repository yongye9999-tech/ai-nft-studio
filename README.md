# 🎨 AI+NFT Studio

> AI驱动的NFT生成与交易平台 | AI-powered NFT Generation & Marketplace

## 项目简介

AI+NFT Studio 是一个去中心化应用（DApp），让用户通过文字描述生成 AI 艺术作品，并一键铸造为 NFT 在链上市场交易。

## 技术栈

- **智能合约**: Solidity 0.8.20 + OpenZeppelin + Hardhat
- **前端**: Next.js 14 + TypeScript + TailwindCSS
- **AI 引擎**: HuggingFace Stable Diffusion + OpenAI DALL-E 3
- **存储**: IPFS (Pinata)
- **区块链**: Ethereum Sepolia / Polygon / BNB Chain
